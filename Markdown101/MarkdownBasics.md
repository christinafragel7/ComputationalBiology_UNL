# Markdown Basics 

**This file is meant to introduce some basic markdown formatting to get you on your way when keeping notes on your projects** 

## Common Syntax:
**Bold Text** = ```**Bold Text**```  
*Italicized Text* = ```*Italicized Text*```  

> This is a block quote = ```> This is a block quote.```   

Code Blocks: 
```code block``` 
or 
```multiline 
code block 
```
